###【紹介】  
いろいろな言語の基礎とも言える言語、  
C言語の基本のお勉強です。  
actionの実行ファイルと対応してます  

###【内容】  
####・HelloWorld.c  
ハローワールドを出力する  
コメントの書き方、おまじない  

####・Variable.c  
変数の使い方、型、出力フォーマット指定子  
  
####・Array.c  
配列  
  
####・String.c  
文字列  
  
####・purs or printf.c  
puts関数とprintf関数の違い  
  
####・if.c  
条件分岐(if文)、比較演算子、比較論理演算子  
  
####・Switch.c  
条件分岐(switch文)  
  
####・for.c  
ループ処理(for文)  
  
####・while.c  
ループ処理(while文)  
  
####・Function.c  
関数  
####・Pointer.c  
ポインタ
